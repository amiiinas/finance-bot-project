# finance-bot-project
### Amanzholova Kamila and Serikova Amina

## Description
Our project is about creating simple financial tracking system. IT was created using Telegram boе and Python(Google colab)
It helps users to track their incomes and expenses also see their balance and create daily and weekly reports 

## Features
- Entering expenses and income  with categories
- Store that data into CSV file for further analysis
- Generate daily and weekly reports

## Requirements 
- Python(Google Colab)
- pyTelegramBotAPI

## Setup instruction
1. Create Telegram bot using BotFather
2. Copy bots token
3. Paste it into yhe code in place of "Token"
4. Install api from requirements and necessary libraries from code in main.py
5. Run the code in Google Colab
6. Start the bot in Telegram

## How It Works
- The user sends a message to the bot
- The bot processes the message
- The data is saved into a CSV file
- The bot calculates income, expenses, and balance

## Limitations
The bot works only while the code is running in Google Colab.
